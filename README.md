# Face_recognition
Проект выполнен на основе проекта https://github.com/akhil838/FREP_scikit-learn/tree/main. Были добавлены 2 модели: SVM и RandomForest.

В проект были добавлены следующие функции:
–  Добавлены две новые модели: SVM, Random Forest.
– Добавлена возможность просмотра датасета: сколько человек (классов) и сколько фотографий у каждого класса.
– Добавлена функции выбора модели.
– Добавлена возможность оценки метрик моделей.
– Добавлена функция выбора гиперпараметров для каждой модели внутри GUI-интерфейса

Для первоначального использования необходимо создать датасет:
1) Кнопка Add Faces открывает окно, где необходимо выбрать камеры из выпадающего списка, ввести имя на латинском и нажать Train. После чего будет создано 100 фотографий
2) Кнопка Go back to Home переключает на домашнюю страницу
3) Для просмотра статистика по датасету, нажать Show Dataset Stats
4) Recognize - для теста модели
5) Compare Models Accuracy - для вычисления метрик моделей
6) Configure Hyperparameters - позволяет выбирать гиперпараметры у каждой модели
7) Запуск процесса обучения модели происходит в момент, когда выбрана камера и нажата кнопка Turn on camera
